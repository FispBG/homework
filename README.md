# homework
Необходимые параметры для программы:
1. ip(-a), т.к. нужен адрес, куда присоединяться.
2. port(-p), т.к. нужен порт, куда присоединяться.
3. id(-i), нужен, чтобы понимать, кто запускает программу.

Опциональные параметры:
1. role(-r), в целом можно задать дефолтное значение, как Client.
2. lib(-l), она может быть по дефолту та, где запускается программа.
